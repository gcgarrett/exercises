# sortNames

Exercise to implement the `sortNames` function that takes a list of names and returns the names sorted by the number of vowels in each name in descending order. If two names have the same number of vowels they are sorted alphabetically.

Exercise comes from [cassidoo's newsletter](https://cassidoo.co/newsletter/), [issue #357](https://buttondown.email/cassidoo/archive/life-is-trying-things-to-see-if-they-work-ray)

## usage
* NodeJS: `node sortNames.js`
* Python: `python3 sortNames.py`

## output
Logs solutions to various lists of names to the console.
